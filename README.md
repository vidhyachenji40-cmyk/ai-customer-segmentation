# AI-Assisted Customer Segmentation

## 🚀 Overview
This project transforms raw retail data into actionable business intelligence. It uses unsupervised machine learning (K-Means Clustering) to group customers by behavior and integrates **Gemini 1.5 Pro** to automatically generate marketing personas and strategies for each segment.

## 🛠️ Tech Stack
- **Language:** Python
- **AI/ML:** Scikit-Learn (K-Means), Google Gemini 1.5 Pro
- **Data:** Pandas, NumPy
- **Infrastructure:** Google Cloud Shell Editor

## 📈 Business Impact
- **Automation:** Replaced manual data interpretation with LLM-driven persona generation.
- **Scalability:** Built a modular pipeline that can handle thousands of customer records.
- **Insight:** Provides instant, natural-language marketing tips for retail managers.
